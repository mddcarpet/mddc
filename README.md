## How to install

Install UDK.
In the UDK installation directory (for example `C:\UDK\UDK-2015-01`), run:

```
git init
git remote add origin https://github.com/filipbartek/mddc.git
git pull origin master
```

## How to run

Run `MDDCGame\play*.bat`.
